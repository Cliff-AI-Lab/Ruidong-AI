# 🎨 内容创作工作室 Content Creation Studio

## Use Case
自媒体/MCN/品牌内容团队: 选题+文案+图/视频/播客/长文全形态生产流水线。

## Agent Composition (榨取 marketing 众多Agent)
```
[Content Creator] ← marketing
[Book Co-Author] ← marketing
[Podcast Strategist] ← marketing
[Short-Video Editing Coach] ← marketing
[SEO Specialist] ← marketing
[Agentic Search Optimizer] ← marketing
[AI Citation Strategist] ← marketing
[Visual Storyteller] ← design
[Image Prompt Engineer] ← design
```

## Bound APIs
| API | Purpose |
|-----|------|
| Unsplash API | 免版权图片 |
| Pexels | 图+视频素材 |
| HuggingFace | 文本/图/视频生成 |
| OpenLibrary | 引用文献 |

## 核心工作流
1. **选题雷达**：热点/搜索/算法
2. **内容生成**：多形态
3. **SEO/AEO优化**：被AI引用+被搜索
4. **分发矩阵**：一稿多投
5. **数据回流**：爆款分析

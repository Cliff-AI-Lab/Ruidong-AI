# 📰 新闻聚合编辑 News Aggregation Editor

## 角色定义
多源新闻聚合与编辑，按主题/地域/情感自动生成日报/周报。

## 核心能力
- 多语种新闻检索
- 主题聚类去重
- 事实核验交叉对比
- 摘要生成

## 绑定API
| API | 地址 | 认证 | 用途 |
|-----|------|------|------|
| NewsAPI | https://newsapi.org/v2 | API Key免费 | 英文新闻 |
| GNews | https://gnews.io/api/v4 | API Key免费 | 多语言 |
| MediaStack | http://api.mediastack.com/v1 | API Key免费 | 新闻聚合 |
| Currents | https://api.currentsapi.services/v1 | API Key | 实时新闻 |

### 调用示例
```bash
curl "https://newsapi.org/v2/top-headlines?category=technology&apiKey=YOUR_KEY"
```

## 工作流程
1. **主题订阅**：标签+源
2. **并行采集**：多API
3. **聚类去重**：同事件合并
4. **摘要生成**：3-5句要点
5. **日报输出**：Markdown/HTML/邮件

# 🏠 房产中介平台 Real Estate Platform

## 应用场景
贝壳/Zillow模式: 房源聚合+成交分析+VR看房+金融服务。

## Agent组合
```
[Real Estate Buyer & Seller] ← specialized
[Geographer] ← academic
[Loan Officer Assistant] ← specialized
[客服智能助手] ← Customer Service
```

## 绑定API
| API | 用途 |
|-----|------|
| Nominatim | 地址解析 |
| OpenRouteService | 通勤时间 |
| ExchangeRate-API | 海外房产 |
| Open Prices | 成交参考 |

## 核心工作流
1. **房源抓取**：开发商+二手
2. **自动估价**：AVM模型
3. **VR看房**：360实景
4. **撮合**：匹配+约看
5. **交易闭环**：贷款+过户

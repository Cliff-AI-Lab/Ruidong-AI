# 🔁 订阅制管理 Subscription Management

## 应用场景
SaaS/媒体/零售订阅业务的定价、试用、续费、挽留。

## 核心能力
- 多层定价
- 试用转化
- 自动续费+催缴
- 流失挽回

## 绑定API
| API | 用途 |
|-----|------|
| Stripe | 订阅支付 |
| Shopify | 实物订阅 |

```
【SaaS订阅业务 月报】
付费订阅: 38,000
MRR: $680K (↑12% MoM)
NRR: 118% (扩展>流失)
套餐:
  - Starter $29 45%
  - Pro $99 38%
  - Enterprise (合约) 17%

  - 试用转付费 22%

  - Gross Churn 2.8%
  - 挽留活动召回 35%流失
```

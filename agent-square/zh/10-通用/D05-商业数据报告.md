# 📊 商业数据报告

## 应用场景
企业战略/市场部的周/月/季度商业报告AI自动生成。

## 典型输出
```
【某公司 月度商业报告】
结构:
  - 市场全景
  - 竞品动态
  - 自家业绩
  - 关键风险
  - 机会洞察
  - 下月计划
AI生成:
  - 数据抓取自动
  - 图表自动生成
  - 观点AI初稿
  - 高管批注
节省: 分析师 每月 20小时
```

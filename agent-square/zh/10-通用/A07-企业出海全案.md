# 🌎 企业出海全案 Enterprise Global Expansion

## 应用场景
中国企业出海东南亚/欧洲/北美/中东的市场准入+本地化+法务+招聘全流程。

## 核心工作流
1. **市场选择**：规模+增速+壁垒
2. **合规准入**：公司注册+牌照
3. **本地化**：产品+定价+传播
4. **人才招募**：本地团队
5. **运营迭代**：KPI+优化

```
【某SaaS出海印度方案】
目标: 印度企业软件市场
市场选择理由:
  - 人口14亿 + GDP 7%增速

  - IT人才储备

  - 传播: LinkedIn为主 + 活动

  - 班加罗尔 总经理+10人团队
  - 起薪 ₹25万卢比(CEO) / ₹8万(工程师)

  - 雇佣合约符合 Indian Labour Code
  - 劳资税+社保按当地
```
